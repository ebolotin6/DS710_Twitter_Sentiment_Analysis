Project Description -
	This is project seeks to answer the question:
		- Are people that talk about fitness happier than people that talk about media (tv, movies, youtube, etc.)?
		- Data source: Twitter data

Directory contents:
	- Project_Summary.pdf - summarizes the project
	- Twitter_Sentiment_Analysis.pdf - Statistics analysis (R markdown PDF)
	- Twitter_Sentiment_Analyzer - directory containing the program code used to complete the project

Directions:
	- Read the project summary to learn about the project.
	- Review the Twitter_Sentiment_Analysis.pdf to review th statistics analysis.
	- Open the TSA folder to review and understand the code. Start with the README.
